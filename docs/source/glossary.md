# Glossary

**GAM**  
General Audience Measurement — BBC’s cross-platform reach framework.

**w/c**  
Week Commencing — Monday starting date of a GAM week.

**Reach**  
Unique individuals exposed to BBC content at least once during a week.

**ServiceID**  
Internal three letter code of BBC editorial services (e.g., WSE, GNL, ARA). The complete list can be visited in GAM_Lookup.xlsx Sheet: ServiceID. The service hierarchies are captured in the same excel sheet in Service Hierarchy

**PlatformID**  
Internal three letter code of each platform (WWW, WIN, WDI, TTK, FBE, ING). The complete list can be visited in GAM_Lookup.xlsx Sheet: PlatformID. The service hierarchies are captured in the same excel sheet in Platform Hierarchy

**PlaceID**  
Three letter country identifier aligned with GAM reporting. The complete list can be visited in GAM_Lookup.xlsx Sheet: CountryID.

**Processed Snapshot**  
The cleaned and enriched dataset ready for aggregation.

**Raw Snapshot**  
Unmodified API response saved weekly from each platform.
